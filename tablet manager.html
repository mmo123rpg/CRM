Page appearance logic

  • When the tablet starts it will start showing the carousel.
  • If the carousel page is turned off, the tablet will show the Menu page.
  • If the Menu page is off, the tablet will show the sign-up page.
  • If the Sing-up page is off, the tablet will show the feedback page.
  • If all pages are off the tabglet will go to the customized page assigned as the ALL OFF PAGE.

Customized pages

A registerd customized page can be assigned to a menu item or as the ALL PAGE OFF link.

The tablet application will not use the time out value or page status. It is here for the developer of the customized page to apply into the customized page logic.

If the customized page is not using the logic a change of the time out value and page status will have no visual or functional effect.
