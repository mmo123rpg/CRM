    Introduction

    Congratulations! You are about to experience the powerful Memberson Shopper Series of products and we will start with a short introduction of the ShopperCRM V6™ which is the powerful CRM platform to manage Shopper Customer Relationship and this introduces all the powerful CRM functionality in detail. ShopperCRM V6™ runs on web platform so it’s easily for users to access anywhere. Below is more detail on each functionalities of the application:

    The Home screen may show different buttons depending on your user right settings. You can navigate between different modules by using the navigation buttons. The Home button takes you back to this start screen.

    1.   Shopper Info

    Personal information of the registered members

    2.   Family Details

    Here family members can be added to a shopper profile and will then usually share the same benefits as the main shopper profile holder. Please note that the system manages family members are differently from Relations (relations links profiles).

    3.   Interests

    This represents interests that the member may have.

    4.  Membership Details

    If you are registering a new shopper as a member, then select the membership type that applies for the shopper. If the shopper is using a sign-up up promotion, then select the appropriate promotion from the Promotion list box and the sign-up promotion privilege will be automatically applied by the system. Most often a sign-up promotion means lowering the Qualification for becoming a member.

    The Remark is a free text field for additional information that is good to know about the registering member. (When a profile is viewed the remark tab shows first if there any remark text written for the Shopper).

    5.  Search Shoppers

    Click the search icon to open the search window. There are three available search options, by Shopper Details, Receipt Details or Special Property Details. Each section is organized in its own tab.

    1.   Tags, Addresses and Contact Preference

    Tags: Categorize and segment shoppers and a shopper can have multiple tags in the profile. Some tags are issued by automation to show if a customer belongs to a specific customer segment. Some tags are issued by system users using the marketing module and in some cases a tag might be given to a specific shopper profile on one by one basis to indicate something, for example black listing a shopper if they have attempted any fraud.

    Addresses: Shopper’s different addresses and contact information is displayed based on the system configuration. Sometimes multiple address options are available.

    Contact Preference: Option of Opt-in/Opt-out Email, and SMS. There might be multiple options available depending on how the system has been configured.

    2.   Shopper Summary, Info, Interests, Password, Remarks and Document

    Summary: The Summary Tab shows important shopper and membership related information like IC number, Date of Birth, Member No, Membership Type, Membership Expiry date, Member Since, Membership Status, etc. IC/passport number, Date of Birth, and Expiry Date can be edited in editing mode.

    Info: The Info Tab holds additional customized shopper data like Nationality, Gender, income range, etc. that may be used for customer segmentation.

    Interests The shopper interests are managed in the interest tab. Marketing and Mailings are usually targeted by shopper interests.

    Remarks The Remark tab is a free text field for any remarks related to the shopper. The remark tab will be showing as the default tab for profiles that have any information in the tab. Often 3trd party systems like an integrated Point of Sale system is configured to show any remark information to the store or service staff.

    Document This tab is to display documents related to particular shopper. Click on an item line to open a document.


    3.  Show/Hide/Remove Personal Data

    Personal data usually includes Phone (H), Phone (O), Mobile, Email, and IC/Passport.

    All this information will be hashed as default. If you have enough rights (E.g., Admin role), then you can show/remove the personal data. Please note that when personal data is removed, it is not possible to reverse.

    4.  Memberships

    All memberships connected with a shopper. A shopper can have multiple memberships connected to their shopper profile. Memberships marked in red color are expired or terminated.

    5.  Activities

    The bottom part of the profile screen displays shows a status bar that list the current situation of total open activities that have to be taken care of.

    The status bar groups activities by activity type. The number of uncompleted activities of each type is written in parentheses. A click on an activity type redirects to the Home page listing all open activities.

    6.  Spending

    The spending Tab lists all purchases for a selected membership. With the ShopperLoyalty™ module each transaction that qualifies for points will also generate a point record. To see which point records that are related to a transaction it is easiest to sort by the date column. Spending totals, transaction numbers, average spending, point balance, points earned, points used and points expiry are visible in the summary section of the spending tab.

    7.  Benefits (only with ShopperLoyalty™ module)

    Discount benefits are listed in the discount tab. Benefits are automatically generated when a membership is created, renewed or upgraded.

    8.  Vouchers

    The voucher tab list all vouchers issued to a shopper. Clicking on a voucher item will enable action buttons at the lower section of the screen, depending on the status of the selected voucher (Issue, Approve, Print, Collect, Utilize, Edit or Refund).

    9.  Communications

    In the communications tab all the communications (Email & SMS) sent to a shopper are listed. The status column shows the status of the communication (Schedule for generation, Sent or Failed).

    10.  Campaigns

    The campaign tab list all campaigns in where the shopper has been targeted. This includes scheduled, running and completed campaigns and the status column shows the current status of each campaign.

    11. Member Renewal History (only with ShopperLoyalty™ module)

    Display all the member renewal historic records in this grid. User will be able to see the Renew Date, Member Number, Valid From and Expiry Date.
